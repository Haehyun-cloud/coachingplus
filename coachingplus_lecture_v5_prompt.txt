코칭플러스 웹사이트의 특강 페이지를 제작한다.

참조 디자인:
- 상단은 짙은 보라색 배경의 페이지 헤더로 구성한다.
- 제목은 "특강", 보조 텍스트는 "Corporate Lecture Programs"로 둔다.
- 글자는 기존 시안보다 약간만 키우고, 과도한 볼드체로 만들지 않는다. 여기서 "진하게"는 글꼴 굵기를 800 이상으로 올리는 의미가 아니라 흐린 흰색/회색 텍스트의 색 대비를 높이는 의미다.
- 흰색 텍스트는 rgba로 흐리게 보이게 하지 말고 가능한 #ffffff에 가깝게 둔다. 본문 회색 텍스트는 너무 연한 회색 대신 #333333 또는 #555555 수준으로 명확하게 보이게 한다.
- "조직의 상황과 대상에 맞게 설계된 근거기반 특강 프로그램입니다. 현장에서 바로 적용 가능한 도구와 실습 중심으로 구성됩니다." 문구는 흰색 독립 박스가 아니라 본문 전체의 같은 연회색 배경 안에 자연스럽게 배치한다.
- 전 직원, 팀 단위, 팀장 & 리더 선택 칸은 더 세련되고 강조되게 만든다. 선택된 탭은 브랜드 보라색 배경과 흰색 글자, 선명한 그림자를 사용한다. 비활성 탭도 카드처럼 보이되 상단 포인트 라인과 hover 상태를 둔다. 단, 탭 글자가 지나치게 크거나 무겁게 보이지 않게 한다.
- 각 탭 아래에는 대상 설명 박스와 세로 카드 목록을 배치한다.
- 각 탭 아래의 대상 설명은 별도 흰색 상자나 라벨 없이 회색 배경 위에 문장만 직접 배치한다. "전 직원 대상", "팀 단위 워크숍", "팀장 & 리더 대상" 같은 라벨은 쓰지 않는다.
- 설명 문장은 한 문장만 사용한다. 예: "구성원 전체가 함께 경험하는 조직문화 특강입니다."
- 카드에는 주제, 상황 설명, 관련 모델, 태그를 포함한다.
- 전체 톤은 코칭플러스의 기존 색상 체계인 #332444, #3e2c50, #4b3765, #f9f9f9, #ffffff를 사용한다.
- 카드 radius는 8px 이하로 유지하고, UI가 너무 장식적으로 보이지 않게 한다.
- 모바일에서는 탭을 세로로 쌓고 글자가 겹치지 않도록 한다.

콘텐츠:
- 전 직원 / 조직문화
  - 심리적 안전감과 조직문화: 구성원이 솔직하게 말하기 어렵고, 실수를 숨기는 분위기가 느껴질 때 / NVC · Positive Leadership / 조직문화, 심리적 안전감
  - 감정소진과 번아웃 예방: 감정 소모가 큰 업무 환경에서 구성원이 점점 지쳐가고 있을 때 / NVC · 자기공감 / 조직문화, 번아웃 예방
  - 변화에 함께 적응하기: 조직 개편이나 새로운 방침 도입 후 구성원의 불안과 저항이 생길 때 / Motivational Interviewing / 조직문화, 변화관리
- 팀 단위 / 워크숍
  - 팀 갈등과 소통 개선: 팀원 간 소통 방식의 차이로 오해가 쌓이고, 협업이 점점 어려워질 때 / DISCplus · NVC / 갈등 관리, 커뮤니케이션
  - 팀 강점 발견 워크숍: 팀원 각자가 어떤 역할에서 강점을 발휘하는지 파악하고 협업 방식을 바꾸고 싶을 때 / StrengthsFinder · VIA / 강점 발견, 팀워크
- 팀장 & 리더 / 리더십
  - 나를 아는 리더십: 리더로서 나의 강점과 성향을 파악하고, 팀에 어떻게 적용할지 고민될 때 / DISCplus · StrengthsFinder / 리더십, 자기이해
  - 피드백과 코칭 대화: 구성원에게 피드백을 주거나 목표를 함께 세우는 대화가 막막하게 느껴질 때 / GROW 코칭 · NVC / 리더십, 커뮤니케이션
  - 구성원의 동기 발견하기: 구성원이 점점 수동적으로 변하거나, 어떻게 동기를 이끌어야 할지 모를 때 / Motivational Interviewing / 리더십, 동기부여
